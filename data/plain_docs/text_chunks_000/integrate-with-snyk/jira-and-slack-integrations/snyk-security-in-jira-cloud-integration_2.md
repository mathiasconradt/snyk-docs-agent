Follow these steps to install Snyk Security in Jira Cloud from the Atlassian Marketplace.

In Jira, navigate to Apps > Find new apps.
Search for Snyk Security in Jira Cloud.\
   If you are installing Jira for EU or AU you must install the dedicated EU or AU app from the Atlassian app marketplace
Click the app and then select Get it now.
Review the information about the app, and select Get it now.
Follow the instructions to install the app.
