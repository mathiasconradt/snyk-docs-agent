The Snyk Learn API endpoints are Beta endpoints.
The Snyk Learning Management add-on offering is in Early Access. For more information, contact your Snyk account team.

The Snyk Learn API endpoints are part of the Snyk REST API and allow for programmatic interaction with Snyk Learn.
The Snyk REST API requires authentication. For information about authentication, see Authentication for API.
