The resulting connection is no longer tied to that specific user, so it is not bound by the userâ€™s access or lifecycle but is linked to the lifecycle of the Organization. The scopes of an App determine which actions an App can perform while connected to a Snyk account. Thescopes are related to actions performed through Snyk Organizations, which determine the API endpoints that can be used in an App. Endpoints authorized through a Snyk Group are not supported. For more information about scopes, see Scopes to request.
