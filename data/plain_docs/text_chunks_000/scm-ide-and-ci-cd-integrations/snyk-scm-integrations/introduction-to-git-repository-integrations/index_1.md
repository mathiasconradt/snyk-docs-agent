Developers can use Snyk to check for issues while writing code locally on their machines before any code changes are pushed to the central Git repository.
Developers can test, fix, and monitor using:

Snyk CLI for local checks on the developer’s machine.\
  Training: see Introduction to the Snyk CLI​.
Snyk for IDEs plugins for checks embedded in the developer IDEs.\
  Training: see Introduction to using Snyk in an IDE.
