To test your repository code using the Snyk CLI, use the snyk code test command.\
For more information, see Scan source code with Snyk Code using the CLI.
Before running Snyk Code tests with the CLI, you may want to do the following:

Update your Snyk CLI version.
Set the Organization for the CLI tests.
Exclude directories and files from the Snyk Code test.

For information about using the snyk code test command and the results, see the pages in this section of the documentation and snyk-to-html.
