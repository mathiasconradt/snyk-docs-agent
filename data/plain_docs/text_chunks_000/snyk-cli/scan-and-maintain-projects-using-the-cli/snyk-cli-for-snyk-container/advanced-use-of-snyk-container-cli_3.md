Frequently used CLI options include:

--json - useful for integrating with other tools
--sarif - useful for integrating with other tools. The option is only available with container test. See also OASIS Static Analysis Results Interchange Format (SARIF).
--exclude-base-image-vulns - only available with container test
--severity-threshold - only available with container test
--exclude-app-vulns
--nested-jars-depth
--fail-on - only available with container test

For more details and CLI options, see the Snyk CLI container help or display the help by running:
snyk container --help
