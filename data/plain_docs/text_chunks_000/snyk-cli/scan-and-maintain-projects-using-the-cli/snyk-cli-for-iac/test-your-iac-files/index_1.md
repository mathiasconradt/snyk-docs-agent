With Snyk Infrastructure as Code, you can test your configuration files with the CLI. This page provides detailed information on how to use certain options for the snyk iac test command.  For information about all of the options see the snyk iac test command help. For details about testing the various configuration files, see the following pages:

Terraform files
CloudFormation files
AWS CDK files
Kubernetes files
ARM files
Kustomize files
Helm charts
Serverless files

In the examples that follow, you can replace the sample file names with the names of your own files, like deployment.yaml.
