Automated provisioning
Provisioning is the first interaction you have with Snyk before getting access to the Snyk platform and features on your plan. For a detailed walkthrough, see the Auto-provisioning guide.
Provisioning can be completed with a new or existing user account and must be completed before you continue the rest of the implementation process.
