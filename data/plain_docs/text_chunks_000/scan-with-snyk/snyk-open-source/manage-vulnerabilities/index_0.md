The documentation in this section explains how to fix vulnerabilities and license issues in your Projects.
To apply fixes, you can use different methods, including manual application in the Snyk Web UI and automatic pull requests opened by Snyk. See Fix your vulnerabilities for more details.
For information about fixability, see Vulnerability fix types. For details, see Upgrade package versions to fix vulnerabilities and Snyk patches to fix vulnerabilities for details.
To help you decide which issues to fix, see Prioritizing and managing issues.
Fixing vulnerabilities relies on the Snyk Vulnerability Database. You may see differences in Open Source vulnerability counts across environments. Information about troubleshooting when you are fixing vulnerabilities is provided.
