To edit an ignore via the Snyk Web UI:

Select the ignored issue from your Organization's Cloud tab.
In the side panel, select the Edit ignore button.
Edit the comment, the ignore expiration date, or both.
Select Confirm.\
   The message "Ignore successfully edited and will be applied on next scan" appears.
To apply the edited ignore immediately, re-scan the environment.\
   You can also wait for the edited ignore to be applied on the next scheduled scan.
After the environment is scanned, refresh the page in your browser to see the edited ignore details.
