The steps follow to onboard a Google Cloud Project to Snyk via the Web UI:

Download an infrastructure as code (IaC) template to give Snyk permissions to scan your Project.
Create a Google service account using the template you downloaded.
Create and scan a Cloud Environment.

When you have completed the steps, you wlll be able to do the following:

View the cloud configuration issues Snyk finds. See Manage cloud issues.
Prioritize your vulnerabilities with cloud context.
