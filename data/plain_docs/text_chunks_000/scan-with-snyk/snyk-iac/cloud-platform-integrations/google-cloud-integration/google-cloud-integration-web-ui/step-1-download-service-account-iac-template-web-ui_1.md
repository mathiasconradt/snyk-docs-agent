In the Snyk Web UI, navigate to Integrations > Cloud platforms.
Select Google Cloud.
In the Add Google Cloud Environment modal, select the Terraform button to download a snyk-permissions-google.tf file:

The Add Google Cloud Environment modal
You can now proceed to Step 2: Create the Google service account.

You can also add a cloud environment from Organization Settings (cog icon) > Cloud environments. See View Environments, Add a Cloud environment.
