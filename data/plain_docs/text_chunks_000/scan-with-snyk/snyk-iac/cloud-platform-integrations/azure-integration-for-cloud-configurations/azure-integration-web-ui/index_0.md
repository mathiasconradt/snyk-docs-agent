The steps follow to onboard an Azure subscription to Snyk via the API:

Download an infrastructure as code (IaC) template or Bash script: to give Snyk permissions to scan your subscription.
Create an Entra ID app registration: using the template or script you downloaded.
Create and scan a Cloud Environment.

When you have completed the steps, you will be able to do the following:

View the cloud configuration issues Snyk finds. See Manage cloud issues.
Prioritize your vulnerabilities with cloud context.
