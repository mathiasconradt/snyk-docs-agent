Navigate to your Snyk Organization, and go to Integrations>GCR.
From the GCR hostname, enter the registry storage region for the images you want to scan in the format region.gcr.io, for example, gcr.io or asia.gcr.io.
Paste the entire contents of the JSON key file you created from Google into the JSON key file field in your Snyk account, as shown in the following screenshot.
Click Save.

Snyk checks the credentials, and when the check is successful, the page reloads with a notification that the connection succeeded.
GCR account credentials
You can now add images to Snyk from GCR.
