To integrate your Snyk Organization with Kubernetes:

On the main page of the Snyk Web UI, navigate to Integrations.
Select Container orchestrators and then select Kubernetes.
Click Connect. On the page that appears, you can find the Integration ID which is created for the Kubernetes integration. The Integration ID is required during the Snyk Controller installation.
Create a Group or Organization service account token. This service account token is required during the Snyk Controller installation. For more information, including the permissions required, see Prerequisites for installing the Snyk Controller.
