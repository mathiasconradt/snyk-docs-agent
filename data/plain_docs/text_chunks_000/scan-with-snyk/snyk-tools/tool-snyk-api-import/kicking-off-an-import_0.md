snyk-api-import supports the same Project sources that you can import using the Snyk API: Git repositories, Docker images, containers, configuration files and much more. You can configure integrations using the Integrations settings on your Snyk Organization settings page. For more information, see the definition of Target on the Snyk Projects documentation page.
Note that any logs will be generated at SNYK_LOG_PATH directory.
