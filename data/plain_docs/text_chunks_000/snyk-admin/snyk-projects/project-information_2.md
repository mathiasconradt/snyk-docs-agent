To view detailed information for a Project, click a Project entry. The page that opens displays the detailed Project information.
The following information is available:

Header: displays a summary of the Project. See Project summary.
Issue cards: display summaries of issues found. See Issue card information.
Views (links at top right):
Overview: displays Project issues, fixes, and dependencies.
History: displays historical snapshots of recent tests. See View Project history.
Settings: displays Project settings.
