In your collection, use the checkboxes to select individual Projects or the select all checkbox to select all visible Projects in a collection. 
You can perform the following actions:
Activate selected Projects.
Deactivate selected Projects.
Add the selected Projects to a collection. In this case, the action will add the selected Projects to another collection that you have created.
Remove selected Projects from a collection. 
Delete selected Projects.
