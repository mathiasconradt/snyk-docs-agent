At the Organization level, select Settings to manage Organization settings:
Organization settings
From the Organization setting menu, select Settings and then select one of the Organization settings options:

General: See Organization general settings for details.
Service Accounts: See Service accounts for details.
Integrations: Set up integrations and see a list of those configured for your Organization; click the name of the configured integration to see the details. See Integrate with Snyk for information about available Snyk integrations.
Open Source: Enable Reachable vulnerabilities analysis and edit language settings; see Snyk Open Source - supported languages and package managers for details.
Snyk Code: Enable Snyk Code; see the Snyk Code documentation for details.
Snyk Iac: Enable Snyk Iac, detecting configuration files, and rules. Select severity levels for configurations scanned. See the IaC documentation for details.
Usage: See the Usage settings page for details.
Notifications: See the Manage notifications page for details.
Snyk See the Snyk Preview page for details.
