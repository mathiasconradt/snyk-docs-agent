On the Policy Manager screen, select Add new policy and in response to the prompts, enter the details.
Enter a policy name and a description to help you quickly identify a policy.\
   Policies in the same category cannot have the same name. You cannot save a policy without a name.
Select whether to apply the policy to Organizations or Project attributes.
Select the Organizations or attributes to which you want to apply the policy.
Add rules to the policy. See Create a license policy and rules or Create a security policy and rules.
Click Submit to create and save the policy.
