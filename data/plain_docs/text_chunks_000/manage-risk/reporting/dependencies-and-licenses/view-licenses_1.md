FieldDescriptionSeverityThe assessed severity level. You can set the severity level in the license policy; see Create a license policy and rules. The default Snyk license policy also defines the Severities for a few licenses.LicenseThe full name of the license. SPDX licenses are linked to the SPDX site, and non-SPDX licenses are linked to their respective sites.If the license is marked as Unknown, Snyk could not find a license for this package.DependenciesThe total number of dependent packages with this license in your Projects, linked to a side panel that displays the full list of affected dependencies in the same layout as the Dependencies tab.ProjectsThe number of Projects with this license, linked to a side panel that displays the full list of your affected Projects, with the same layout as the Dependencies tab.
