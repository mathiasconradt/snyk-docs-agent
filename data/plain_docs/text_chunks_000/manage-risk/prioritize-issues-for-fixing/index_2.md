Prioritize the issues within a Project by using any of the following custom filters: 

Issue type
Severity 
Fixability
Exploit Maturity

You can use this type of prioritization with any of the following Snyk Plans: 

Snyk Free/Team Plan
Snyk Enterprise Plan
Snyk Essentials
Snyk AppRisk 

You can prioritize your Projects at the Snyk Organization level.
